Aipplica Spectrometry Solutions.

Aipplica Systems designs and builds Embedded Optoelectronic System Solutions. We make high quality, robust, handheld Spectrometers & Spectrophotometers & accessories to realize a complete solution for customer application. Spectrometer User interface software supports various analyses on captured spectrum such as measuring irradiance, absorbance, reflected spectrum, chemical constituent analysis and more. Captured spectral data is provided as plain text file, giving user complete flexibility for further spectrometric analysis by 3rd party tools. We customize our devices to customer requirements by selecting and designing with the most appropriate components and subsystems for the application.

Small sized APS-1910 spectrometer plugs into USB-C port and a User friendly interface software makes easy work of capturing and analyzing spectra.
